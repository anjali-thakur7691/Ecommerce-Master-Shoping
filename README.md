A sleek, responsive, and modern e-commerce web application built using Python (Flask) and SQLite. This platform features robust user authentication, dynamic product cataloguing, and an intuitive shopping cart management system.

---

## ✨ Core Features

* **🔐 User Authentication:** Secure registration and login functionalities for users.
* **🛍️ Product Catalog:** Visually appealing display of electronic products and gadgets with real-time prices and descriptions.
* **🛒 Shopping Cart:** Seamless 'Add to Cart' functionality, quantity updates, and dynamic checkout system.
* **👤 Dynamic Dashboard:** Personalized profile view displaying user details and historical orders after a successful login.

---

## 🛠️ Tech Stack

* **Backend:** Python, Flask
* **Database:** SQLite (`database.db`)
* **Frontend:** HTML5, CSS3, Jinja2 Template Engine, Bootstrap, jQuery
